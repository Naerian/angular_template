Tipos de barras de progreso
El componente neo-progress-bar es utilizado para mostrar el progreso de una tarea o acción en curso. Este componente es útil para indicar al usuario que una acción está en curso y que debe esperar. La barra de progreso puede tener diferentes tamaños, colores y estilos. También puede mostrar el porcentaje de progreso y dividirse en dos partes para mostrar dos valores diferentes.
